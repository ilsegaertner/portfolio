Meet App

A Progressive Web App for Discovering Tech Events by City

React · AWS Lambda · OAuth · TDD · D3.js · Service Workers

What It Is

Meet App is a serverless, mobile-friendly React application that lets users browse upcoming tech events in cities around the world. The app connects to the Google Calendar API using OAuth2 authentication and allows users to search by location, view event details, and visualize data through charts.

What I Did

I built the app using React with a focus on progressive enhancement and test-driven development (TDD). I implemented OAuth2-based authentication, offline functionality via service workers, and modular charts for data visualization. The app supports installation as a standalone PWA on mobile and desktop.

Key Features
  • City-based event filtering and keyword search
  • Interactive charts showing event distribution (D3.js)
  • OAuth2 login flow via Google Calendar API
  • Offline access and installable PWA functionality
  • Fully tested (90%+ coverage) with unit and integration tests
Why

This was a technically demanding project where I applied core software engineering practices like TDD and service abstraction. It taught me how to build applications that are resilient, user-friendly, and performant — even with unreliable network conditions.

What I took away
  • Implementing TDD in React using Jest and Enzyme
  • Managing third-party OAuth flows securely in a frontend app
  • Designing offline-first features with caching and service workers
  • Using D3.js to make complex data readable and interactive
