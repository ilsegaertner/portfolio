World Weather App

Responsive React App for Real-Time Global Weather and Forecasts

React · OpenWeatherMap API · Radix UI · Leaflet · Framer Motion

What It Is

The World Weather App is a responsive weather dashboard built with React. It allows users to search for any city worldwide and instantly view current weather conditions, a five-day forecast, live icons, and an interactive map — all within a visually adaptive interface that changes based on the weather.

What I Did

I built the app entirely in React, integrating the OpenWeatherMap API for real-time data and Leaflet for map rendering. I also designed and implemented a dynamic background system where the UI adapts visually to match current conditions (e.g. sunny, stormy, nighttime), using Framer Motion to create smooth transitions and a polished experience.

Key Features
  • City search with current and 5-day weather forecasts
  • Live weather icons and real-time temperature display
  • Interactive maps for geographic context (Leaflet)
  • Animated weather-based background gradients
  • Chrome browser extension for quick access
Why

This project challenged me to work with asynchronous data, API rate limits, and loading states — all while maintaining a fluid and engaging user experience. It also pushed me to think beyond functionality and explore how microinteractions and animations can improve usability and retention.

What I took away
  • How to structure API calls and manage loading/error states cleanly in React
  • How to animate UI elements meaningfully (not just for flair)
  • How to design with real-world variability in mind — e.g. slow connections, invalid city inputs, dynamic light/dark visuals
