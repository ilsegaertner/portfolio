CUB Film Data (React)

Full-Stack Movie App Built with the MERN Stack

React · JavaScript · MongoDB · Express · CSS

Case Study

Overview

CUB Film Data is a full-stack web application that lets users explore arthouse films, view details about directors and genres, and manage a personal list of favorite movies. I developed both the backend REST API and the client-side React interface, integrating secure user authentication and responsive UI components. The project was built as part of my Full-Stack Web Development course and later refined for inclusion in my professional portfolio.

Objective

The goal was to demonstrate full-stack proficiency using the MERN stack (MongoDB, Express, React, Node.js). I aimed to build a fully functional app from scratch — with real-world features like authentication, data validation, client-server communication, and deployment — while writing clean, modular code.

Problem

How do you design and build a scalable, user-friendly movie database from scratch — handling everything from user authentication to dynamic UI updates — with no starter code or templates? I needed to:

  • Create a robust backend that could handle multiple user operations
  • Build a responsive, intuitive frontend from scratch in React
  • Ensure secure, seamless communication between both layers
  • Learn and apply new technologies like JWT, Mongoose, and React Router efficiently
Approach

Backend

  • Built a REST API using Node.js and Express
  • Designed and tested endpoints (GET, POST, PUT, DELETE) using Postman
  • Integrated MongoDB and Mongoose for data modeling and storage
  • Implemented JWT-based authentication for login and protected routes
  • Deployed backend to Heroku and MongoDB Atlas

Frontend

  • Developed the interface in React with React Router for navigation
  • Built reusable components: movie cards, modals, forms, and profile views
  • Connected frontend to backend via Fetch with token-based headers
      • Enabled users to:
    • Register, log in, and update their profiles
    • Browse movies and view detailed info (genre, director)
    • Add/remove favorite films
  • Styled using custom CSS with a focus on responsiveness
Challenges

Building the backend taught me how to manage data security and API logic, but the frontend was where I encountered the steepest learning curve. I had to:

  • Understand how to manage state effectively with props and conditional rendering
  • Learn React from the ground up — especially routing, lifting state, and error handling
  • Debug connection issues between frontend and backend
  • Build and test a full login flow using real credentials and token handling

These challenges pushed me to become a more autonomous problem-solver and gave me a much deeper grasp of JavaScript fundamentals.

Results
  • Successfully deployed a working full-stack application with user login, persistent data, and dynamic UI
  • Developed a stronger understanding of how APIs and frontend components interact
  • Learned to debug full-stack workflows independently
  • Today, I would improve code modularity and refactor the component structure for scalability
