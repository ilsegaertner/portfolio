CUB Film Data Angular

An Angular App for Exploring Arthouse Films

React · JavaScript · MongoDB · Express · CSS

What It Is

CUB Film Data is a single-page web app that allows users to browse arthouse films, learn more about directors and genres, and manage their user account. It connects to a REST API I built previously and serves as a frontend interface for movie exploration.

What I Did

I built the entire frontend using Angular and TypeScript. I structured the app into modular components — including a main view, movie cards, modals for director/genre details, and a profile section. I also handled routing, form validation, and API integration.

Key Features
  • User registration, login, and profile updates
  • Movie browsing with detailed views (using Angular modals)
  • Filtering/search functionality
  • Account management (including delete account)
Why

This was my first major project using Angular after working in React. I quickly saw the strengths of Angular’s architecture — particularly its separation of concerns and use of services. I also learned to work more systematically: planning user flows with stories, organizing SCSS using design tokens, and thinking in reusable UI modules.

What I took away

This project deepened my understanding of TypeScript and client-side architecture. It also taught me to balance structure and flexibility — not just in code, but in UX. Most importantly, it gave me confidence in building real-world apps from scratch.
