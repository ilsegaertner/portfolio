CUB Film Data Angular

An Angular Application For An Arthouse Movie Database

Back

CUB Film Data (Angular), an Angular-based project, stands as the frontend interface of a movie database (which I built previously as REST-Api for the backend).

Users can browse through movies, get data about directors and genres, or edit their profile.

For the project I learnt Typescript and Angular which I found easier than React, generally structured and pleasant to work with.

Prior to implementation, user stories helped me to visualize necessary interfaces and routes (see link), aiding in the understanding of required operations and serving as a helpful visualization tool.
