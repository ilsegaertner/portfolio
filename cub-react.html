CUB Film Data (React)

React Frontend for a Full-Stack Movie Database Application

React · JavaScript · MongoDB · Express · CSS

What It Is

CUB Film Data is a single-page React application that connects to a custom-built REST API to display and manage an arthouse movie database. Users can register, log in, browse and search films, view detailed information, and manage a list of favorite movies.

What I Did

I built the entire client-side using React and CSS, connecting it to a backend I had previously developed with Node.js, Express, and MongoDB. I implemented user authentication, RESTful communication, and dynamic filtering of data. The app also allows users to update their profile or delete their account.

Key Features
  • Full user account management (signup, login, edit profile, delete account)
  • Filterable movie catalog with detail views
  • Favorite movies functionality with persistent user data
  • Responsive UI for desktop and mobile
  • Communication with a secure REST API
Why

This project represents the integration of both frontend and backend logic — demonstrating my ability to plan and build full-stack applications. It challenged me to structure client-server communication cleanly, handle protected routes, and manage persistent user state.

What I took away
  • Building scalable components in React using hooks and conditional rendering
  • Handling asynchronous API requests and responses with user feedback
  • Structuring backend endpoints for frontend consumption
  • Thinking through UX flows from signup to long-term use (e.g. editing favorites)
